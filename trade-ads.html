Roadmap Note

Making this board shared across visitors

Right now trade ads are stored only in your own browser (via localStorage), so this works as a personal scratchpad but two different visitors won't see each other's posts. To make it a real shared board, this page needs a small backend — a database table for ads plus an API to create/list/expire them (e.g. Supabase, Firebase, or a lightweight Postgres service). That's the natural next build step once the rest of the site is live.
